

# FAU Erlangen Master's Programs Query Interface

This project is a Python-based application that scrapes information from FAU Erlangen's master's program web pages, processes the data into a vector store using OpenAI embeddings and Chroma, and provides a Gradio interface to query information about these programs using a Retrieval-Augmented Generation (RAG) system.

## Features
- **Web Scraping**: Extracts content from specified FAU Erlangen master's program pages.
- **Data Processing**: Converts scraped data into documents, splits them into chunks, and stores them in a Chroma vector store with OpenAI embeddings.
- **Query Interface**: Uses a Gradio interface to allow users to ask questions about FAU master's programs, with responses generated by OpenAI's `gpt-3.5-turbo` model based on retrieved documents.
- **Secure Configuration**: Loads sensitive information (e.g., OpenAI API key) from a `.env` file.

## Prerequisites
- Python 3.8 or higher
- An OpenAI API key (obtainable from [OpenAI](https://platform.openai.com/))
- Internet connection for web scraping and API calls

## Installation

1. **Set Up a Virtual Environment** (recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

2. **Install Dependencies**:
   Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. **Set Up Environment Variables**:
   Modify `.env` file in the project root directory with the following content:
   ```
   OPENAI_API_KEY=your-actual-openai-api-key
   ```
   Replace `your-actual-openai-api-key` with your OpenAI API key.

4. Run source.py
